Welcome to Silpa!

Silpa, Swathanthra Indian Language Processing Applications is a web platform to host the free(dom) software language processing applications easily.It is a web framework and a set of applications for processing Indian Languages in many ways. Or in other words, it is a platform for porting existing and upcoming language processing applications to the web. Silpa can also be used as a python library or as a webservice from other applications. Silpa is in development and if you are interested in contributing contact the developers
